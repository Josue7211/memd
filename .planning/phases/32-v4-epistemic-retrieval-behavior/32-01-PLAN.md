# Plan 32-01: `v4` Epistemic Retrieval Behavior

## Goal

Make retrieval behavior prefer verified evidence over narrative continuity.

## Steps

1. Add an epistemic ranking adjustment to context retrieval.
2. Apply the same epistemic adjustment to search ranking.
3. Weight canonical and recently verified memory upward.
4. Penalize synthetic and unverified memory when all else is equal.
5. Add regression coverage showing verified canonical memory outranks unverified synthetic memory.

## Verification

- `cargo test -q -p memd-server`
- `cargo test -q`

## Result

`memd` retrieval becomes more truth-seeking by default instead of only more
convenient.
