# Plan 31-01: `v4` Automatic Short-Term Memory Management

## Goal

Capture meaningful short-term coordination state automatically.

## Steps

1. Factor a reusable auto-checkpoint helper for bundle event capture.
2. Trigger automatic short-term capture on claim acquire/release/transfer.
3. Trigger automatic short-term capture on assignment, help, review, and other peer message sends.
4. Keep captured content compact and event-shaped instead of transcript-shaped.
5. Refresh bundle memory files after automatic capture.
6. Verify the client suite and full workspace tests.
