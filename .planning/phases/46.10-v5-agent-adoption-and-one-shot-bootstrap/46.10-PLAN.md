# Phase 46.10: `v5` Agent Adoption and One-Shot Bootstrap — Plan

## Goal

Make Codex, Claude, and OpenClaw bootstrap memd automatically so the user
doesn't have to keep re-importing the same project context.

## Plan

1. Keep the startup hooks aligned with memd for the main runtimes.
2. Ensure bootstrap reads the existing memory sources once.
3. Keep the user-facing flow simple and one-command obvious.
4. Treat memd as the shared source of truth after startup.

## Verification

- `cargo fmt --all`
- runtime hook checks for Codex and Claude
- OpenClaw workspace bootstrap docs reference memd

## Result

The common agent runtimes start from the same shared memory substrate.
