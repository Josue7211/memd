# Phase 47.4: `v6` Universal Design Memory — Plan

## Goal

Make reusable design guidance a typed memory lane that can move across
sessions and harnesses without being re-explained.

## Plan

1. Import design artifacts as first-class project memory.
2. Classify them distinctly in the source registry.
3. Keep the design lane inspectable and portable.
4. Preserve harness-aware metadata for future routing.

## Verification

- `cargo fmt --all`
- `cargo test -p memd-client`
- `DESIGN.md` and `design/*.md` are part of bootstrap source collection

## Result

Design guidance becomes reusable memory instead of disposable prompt text.
