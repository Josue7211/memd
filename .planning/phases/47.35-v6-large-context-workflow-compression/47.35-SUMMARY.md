# Phase 47.35: v6 Large-Context Workflow Compression — Summary

## Goal

Keep huge jobs workable without making every intermediate turn huge.

## What Changed

1. The layered workflow is explicit.
   - global brief
   - glossary / terminology memory
   - entity and reference sheets
   - chunk-local working windows
   - reconciliation / harmonization passes
2. Broad context is deliberate.
   - the workflow preserves global coherence without giant context every turn
   - provenance and cross-chunk traceability stay intact
3. Short and medium sessions get better.
   - repeated context is compressed into reusable layers
   - only the passes that truly need broad context pay for it

## Verification

- roadmap already captures the layered long-form workflow

## Result

Memd can make short and medium sessions competitive with huge sessions for
most long-form work while still supporting deliberate broad-context passes.
