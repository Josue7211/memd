# Plan 42-01: `v5` Coordination Watch and Alert Views

## Goal

Make coordination pressure easier to monitor during active coworking by adding
compact watch and alert views on top of the current coordination dashboard.

## Steps

1. Add a watch mode to the CLI coordination surface with bounded refresh output.
2. Reuse the existing view categories so watch mode can focus on one slice or the full dashboard.
3. Add compact alert lines for meaningful pressure changes instead of replaying full state every time.
4. Keep the first slice local and bounded before considering richer UI surfaces.

## Verification

- operators can keep coordination pressure visible without manually rerunning the same command
- watch output remains compact instead of transcript-like
- alert lines stay aligned with the same coordination categories used by dashboard and drilldown views

## Result

Coordination moves from static inspection toward lightweight live monitoring.
