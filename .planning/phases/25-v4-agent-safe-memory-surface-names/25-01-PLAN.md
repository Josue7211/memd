# Plan 25-01: `v4` Agent-Safe Memory Surface Names

## Goal

Prevent bundle memory-file collisions with agents that already own `MEMORY.md`.

## Steps

1. Change the shared bundle root memory file from `MEMORY.md` to `MEMD_MEMORY.md`.
2. Leave agent-specific files such as `CODEX_MEMORY.md` and `CLAUDE_CODE_MEMORY.md` intact.
3. Update README and integration docs to reference the new shared root filename.
4. Update tests that read the root memory placeholder.
5. Record the phase completion in `.planning`.

## Verification

- `cargo test -q -p memd-client`
- `cargo test -q`

## Result

`memd` owns its own memory surface without colliding with native agent memory
conventions.
