# Phase 47.3: `v6` Raw-to-Graph Compilation for Knowledge Workspaces — Plan

## Goal

Transform raw workspace folders into reusable compiled knowledge artifacts so
repeated lookup can prefer compiled graph outputs over raw rereads.

## Plan

1. Keep the compiled lane filesystem-first.
2. Preserve provenance classes on graph edges.
3. Prefer compiled outputs when fresh enough.
4. Keep the path durable and reusable across sessions.

## Result

Medium-scale workspaces can answer from compiled knowledge artifacts first.
