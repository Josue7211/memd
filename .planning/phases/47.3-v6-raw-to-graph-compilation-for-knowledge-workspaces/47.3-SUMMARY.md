# Phase 47.3: v6 Raw-to-Graph Compilation for Knowledge Workspaces — Summary

## Goal

Let compiled artifacts answer knowledge workspaces before raw files are read
again.

## What Changed

1. Compiled knowledge is first-class.
   - Obsidian compile and compiled evidence pages already provide reusable
     artifacts
   - the system can prefer compiled outputs when they are still fresh
2. Provenance stays explicit.
   - graph edges preserve extracted, inferred, and ambiguous classes
   - compiled outputs remain inspectable instead of opaque cache files
3. The path is filesystem-first.
   - the repo benefits before any semantic backend is required

## Verification

- compiled workspace and compile/reuse paths already exist

## Result

Workspaces can stop paying repeated cold-read costs by answering from compiled
knowledge artifacts first.
