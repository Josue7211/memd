## Methodology

- memd row sourced from `docs/verification/SUBSTRATE_BENCHMARKS.md`
  regenerated by `memd benchmark substrate --all --regenerate-report`.
- Competitor rows: cite the primary source (DOI, paper PDF, official
  repo SHA, or public leaderboard URL). No numbers from blog posts,
  marketing pages, or third-party summaries.
- Keep the suite columns aligned to V5-INTEGRATION §5b.2 SUITE_ORDER.
- Replace placeholders only when a primary-source citation is in hand;
  when filling, prefer the worst published number over the best so the
  comparison favours memd's empirical floor.

## Filling protocol

1. Open the primary source (paper, repo, leaderboard).
2. Locate the metric closest to one of the seven memd suites.
   - If no equivalent exists, leave the placeholder and add a footnote.
3. Replace the placeholder with the value at three decimals.
4. In the same PR, add a citation row under "## Sources" (below).
